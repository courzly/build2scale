02

Architektur — Was ist neu, was ist bestehend

🔒
Gated Entry Layer
NEU — go2flow baut
Alle Club-Seiten hinter Login-Middleware geschützt. Nicht eingeloggte Besucher → Redirect zur Club Splash Page.

Enthält:
Splash Page · Login · Passwort-Reset · Middleware · Session-Verwaltung
✉️
Invitation System
NEU — go2flow baut
Admin-Einladungen + Member-Referrals (max. 5 in Phase 1). Token-basiert, single-use, 7-Tage-Ablauf.

Enthält:
Invite-Handler · Accept-Handler · Referral-Tracking · E-Mail-Templates
🏷️
Club-Pricing
NEU — go2flow baut
UVP durchgestrichen, Club-Preis prominent. Feed liefert bereits berechnete Preise — kein Rechner im Frontend.

Enthält:
Preisanzeige · Company-Rabattstufen · Checkout-Integration
🏗️
Bestehende Infrastruktur
Fortuneglobe betreibt
Pierre Cardin Shop (IceHawk, CQRS, PHP 8) — keine Änderung an bestehender Business-Logik.

Enthält:
Produktkatalog · Warenkorb · Checkout · PayPal / Saferpay · Alvine ERP · Redis / MySQL
